shardcache_client_get_aync_data_cb  cb,
void *  priv 
)

Get the value for a key asynchronously.

Parameters
cA valid pointer to a shardcache_client_t structure
keyA valid pointer to the key
klenThe length of the key
cbThe shardcache_client_get_aync_data_cb which will be called for each received chunk
privA pointer which will be passed to the shardcache_client_get_aync_data_cb at each call
Returns
0 on success, -1 otherwise
Note
This function will block and call the provided callback as soon as a chunk of data is read from the node. The control will be returned to the caller when there is no more data to read or an error occurred
